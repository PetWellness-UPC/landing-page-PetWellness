Acerca de

Somos una iniciativa dedicada a promover el bienestar y la salud de su mascota. Aportando informacion y cuidados esenciales.

Misión

Nos dedicamos a promover el bienestar de las mascotas a través de información confiable y servicios veterinarios de alta calidad, con el objetivo de mejorar su calidad de vida y fortalecer el vínculo con sus dueños.

Visión

Buscamos liderar una comunidad comprometida con el bienestar de las mascotas. Nuestra visión es un mundo donde todas las mascotas reciban amor, cuidado y atención para una vida feliz y saludable.

Galeria de imagenes

Pantalla de inicio
La pantalla de inicio es la primera interfaz que se presenta al usuario al acceder a un sitio web o aplicación móvil. Su diseño y funcionalidad son cruciales para crear una primera impresión positiva y guiar al usuario hacia las acciones deseadas.

Perfil de usuario:
El perfil de usuario es una sección de un sitio web o aplicación móvil donde los usuarios pueden ver y editar su información personal

Listado de clientes para el veterinario:
El listado de clientes para el veterinario es una sección de un software de gestión veterinaria donde se puede ver un registro de las consultas y los pacientes atendidos.

Formulario de reserva
En esta sección puedes ingresar los datos para realizar una reserva de consulta veterinaria. Deberás seleccionar la fecha y hora disponibles, tanto para ti como para el veterinario.

Chat en Linea
En esta sección, los usuarios pueden acceder a nuestro servicio de chat en línea para comunicarse directamente con un veterinario.

En el movil
Versión Móvil Nuestra plataforma cuenta con una versión optimizada para dispositivos móviles, brindando a los usuarios acceso ágil y conveniente a todos nuestros servicios. con la diferencia que la barra de inicio no se muestra hasta que selecciones el icono de rayas verticales.

Codigo:
Este código proporciona la base estructural para la página web de PetWellness, utilizando HTML, CSS e imágenes. El equipo de desarrollo lo está implementando por etapas.

Github:
La página web de PetWellness se desarrollará mediante un proceso colaborativo en GitHub. Cada integrante del equipo asignará una sección específica del código y la implementará de forma individual. Posteriormente, el código individual de cada miembro se subirá a GitHub para su integración en el proyecto final.

Jira:
La herramienta Jira se utilizará para que cada miembro del equipo del proyecto de desarrollo de la página web de PetWellness registre sus avances y tareas completadas. Esto permitirá un seguimiento centralizado del progreso individual y facilitará la colaboración entre los miembros del equipo.
